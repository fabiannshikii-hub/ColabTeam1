X = Satu produk Mug ekoprint
y = Satu Produk Tempat sampah
z = Satu produk Lampu

Model Matematika
x + y + z = 125.000.....1
x + 2y + z = 185.000.....2
2x + y + 3z = 155.000.....3

Penyelesaian
•Eliminasi => y pers 1 dan 2
x + y + z = 125.000 /×2/ 2x + 2y + 2z = 250.000
x + 2y + z = 185.000 /×1/ x + 2y + z = 185.000-
x + z = 65.000...4

•Eliminasi y => 1 dan 3
x + y + z = 125.000
2x + y + z = 155.000-
-x = -30.000...50

•Eliminasi pers 4 dan 5
x + z = 65.000
-x = -30.000+
z = 35.000

•Substitusi z = 35.000 ke pers 4
x + 35.000 = 65.000
x = 65.000 - 35.000
x = 30.000

•Substitusi z = 35.000, x = 30.000 ke pers 1
x + y + z = 125.000
30.000 + y + 35.000 = 125.000
y = 125.000 - 30.000 - 35.000
y = 60.000
